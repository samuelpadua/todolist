# To Do List

Este projeto foi desenvolvido com React e React-Redux


**Desenvolvimento**

```sh
npm install && npm start
```
**Testes**

```sh
npm test
npm run test:watch
npm run test:cov
```

**Build e deploy**
```sh
npm run deploy
```
Após termino do script, o projeto estará disponível em https://samuelpadua.github.io/todolist/
